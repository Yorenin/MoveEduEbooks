# MoveEduEbooks
Esse é um projeto para site de ebooks da faculdade.

### Como executar

1. Clone este repositório.
2. Abra a pasta no Visual Studio Code.
3. Certifique-se de que os arquivos `index.html`, `style.css`, `script.js` e as imagens estejam todos juntos na mesma pasta.
4. Clique duas vezes no `index.html` para abrir no navegador.
5. Interaja com o catálogo: adicione livros ao carrinho, visualize descontos e demais animações.
